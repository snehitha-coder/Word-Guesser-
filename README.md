# Word Guessing Game (Python)

## Description
This is a simple command-line based Word Guessing Game built using Python.  
The program randomly selects a word, and the player must guess the word by entering letters within a limited number of attempts.

This project helps in understanding basic Python concepts such as loops, conditionals, strings, lists, and user input handling.

---

## Features
- Random word selection
- Letter-by-letter guessing
- Limited number of attempts
- Displays progress of the guessed word
- Simple and user-friendly interface (CLI)

---

## Tech Stack
- Language: Python
- Platform: Command Line Interface (CLI)

---

## How to Run the Project
1. Make sure Python is installed on your system
2. Download or clone this repository
3. Open a terminal in the project folder
4. Run the file using:
